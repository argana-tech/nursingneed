# A項目

・対象データ

EFファイルの「レセプト電算コード(9列目)」がA項目マスタのコードと一致するものを対象として
処理を行います。

・比較処理

以下全てに当てはまるEFファイルのレコードが存在する場合、正常として表示します。

 - EFファイルの「識別番号(2列目)」     ＝　Hファイルの「識別番号(3列目)」
 - EFファイルの「実施年月日(24列目)」  ＝　Hファイルの「実施年月日(6列目)」
 - Hファイル「コード(列目)」         ＝　「ASS0010」
 - EFファイルの「レセプト電算コード(9列目)」とHファイルの「ペイロード番号(10-17列目)」が一致（紐づけはAマスタを使用）（※1 ペイロードと電算コードのチェック処理について）

当てはまらない場合は、チェック漏れとして扱います。


# C項目

・対象データ

EFファイルの「レセプト電算コード(9列目)」がC項目マスタのコードと一致するものを対象として
処理を行います。

・比較処理

以下全てに当てはまるEFファイルのレコードが存在する場合、正常として表示します。

 - EFファイルの「識別番号(2列目)」     ＝　Hファイルの「識別番号(3列目)」
 - EFファイルの「実施年月日(24列目)」  ＝　Hファイルの「実施年月日(6列目)」
 - Hファイル「コード(列目)」         ＝　「ASS0030」
 - EFファイルの「レセプト電算コード(9列目)」とHファイルの「ペイロード番号(10-16列目)」が一致（紐づけはCマスタを使用）（※1 ペイロードと電算コードのチェック処理について）

当てはまらない場合は、チェック漏れとして扱います。



# 説明

※1 ペイロードと電算コードのチェック処理について

1. 電算コードをキーとして、A/Cマスタからペイロード番号を取得
2. Hファイルより、取得したペイロード番号の値をチェック
3. 2で取得した値が0以外は一致すると判断
